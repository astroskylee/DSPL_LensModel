MTF 公式使用 PixInsight 同一 midtones transfer function。`m = 0.5` 为线性；`m < 0.5` 提亮中间调；`m > 0.5` 压暗中间调。

How To Use
  • 单击添加一个顶点。
  • 双击当前画布，或按 Enter，闭合当前 polygon。
  • 按 Backspace 撤回当前 polygon 的最后一个点。
  • 切换到 Conjugate Point Mode 后，单击图像会记录一个共轭点坐标。
  • 切换到 Trace Point Mode 后，单击图像会记录一个 trace point，并用数字编号显示。
  • Save Conjugate JSON 会优先打开浏览器保存对话框，方便你直接保存到 preprocessed 目录。
  • Save Trace JSON 会导出 trace point 的 display/FITS 两套坐标，文件名以 _trace_points.json 结尾。
  • 如果加载的是 FITS，导出的 mask FITS 会和当前显示的图像尺寸一致，并会自动做回 y 轴翻转，保证和原始 FITS 像素网格对齐。
